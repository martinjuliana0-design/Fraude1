# Fraude1
---
## Descripción del proyecto
---
Este proyecto tiene como objetivo analizar un conjunto de transacciones financieras con el fin de comprender el comportamiento del fraude desde una perspectiva descriptiva. A través del uso de SQL Server Management Studio para la exploración de datos y Power BI para la visualización, se identifican patrones temporales y económicos asociados a las transacciones fraudulentas.
El análisis está orientado a un enfoque analítico y exploratorio (EDA), sin aplicar modelos predictivos, simulando el trabajo de un Data Analyst junior en un contexto real de negocio.
---
## Objetivo del proyecto 
---
Analizar el comportamiento de las transacciones fraudulentas para identificar patrones temporales y diferencias en los montos respecto a las transacciones no fraudulentas.
